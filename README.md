

## Student Portfolio Website

This is a simple multi-page student portfolio website created using HTML and CSS.
It was developed by Ritik as part of a web development assignment under the guidance of faculty member Anurag Sir.

Project Pages:

1. Home Page
2. About Me Page
3. Skills and Projects Page
4. Contact Page

Technologies Used:

* HTML
* CSS

Features:

* Multi-page layout
* Navigation bar available on every page
* Skills displayed using card layout
* Simple contact form (user interface only)

Project Structure:
index.html
about.html
skills.html
contact.html
css/style.css
images/profile.jpg

How to run:
Download the files and open index.html in any browser.
